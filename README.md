<h1 align="center">
  Barber Shop
</h1>

Projeto full stack que implementa o agendamento de clientes em uma Barbearia.

## Tecnologias

- back-end
  - Java 21
  - Spring Boot
  - Spring MVC
  - Spring Data JPA
  - Docker
  - PostgreSQL
  
- font-end
  - TypeScript
  - Angular
  - Angular CLI
  - Angular Material
  - Bootstrap
  - ngx-mask
  - HTML
  - SCSS

## Práticas adotadas

- API REST
- Consumo de API Rest
- Componentização
- Consultas com Spring Data JPA
- Injeção de Dependências
- Tratamento de respostas de erro
